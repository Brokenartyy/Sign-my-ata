## üìñ About

**Sign My ATA** is a personal guestbook-style web project inspired by atabook.org,  
but built **from scratch** without using any existing templates.

This project is hosted on **GitHub Pages** and connected to a **cloud database (Firebase)**.

The main goal is simple:
- Full control over the ATA
- Freedom to update anytime
- No dependency on third-party templates

Reality check: control comes with pain.
